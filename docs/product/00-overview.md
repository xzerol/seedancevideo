# 产品总览

Seedance Video 是一个本地 Docker 化的 AI 视频/图片生成工作台。

核心能力：

- 项目首页显示最近 10 个项目，可新建、删除、进入项目；更多项目进入分页列表。
- 通过节点画布组织素材、人物、图片生成、视频生成和结果复用。
- 使用 TOS 保存上传素材，让火山方舟可以读取输入资源。
- 使用 Seedream 5.0 生成图片，使用 Seedance 2.0 生成视频。
- 项目、画布布局、连线、素材引用和生成历史保存在本地 SQLite。
- 生成结果默认留在当前画布，用户手动收藏后进入全局资源库。

当前访问地址：

```text
http://localhost:3001
```

主要用户路径：

1. 在项目首页查看最近项目，或进入分页列表查找历史项目。
2. 新建或打开项目。
3. 从人物角色库/素材库上传或拖拽资源到画布。
4. 创建 Seedream、Seedance 或百炼视频生成节点。
5. 通过连线把资源或结果接入生成节点。
6. 生成图片/视频，并把结果继续连到下游节点。
7. 需要长期复用的结果点击“收藏到资源库”。
