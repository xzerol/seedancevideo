# 资源库

左侧栏包含三个资源入口：人物角色库、素材库、历史视频。

每个入口都支持折叠。展开时列表最多显示 5 条高度，超出后在入口内部滚动，避免左侧栏过长影响画布操作。

## 人物角色库

用于管理人物参考图。

人物角色库是全局共享资源，不跟随项目隔离。不同项目可以复用同一批人物角色。

能力：

- 上传人物图片。
- 搜索人物。
- 拖拽到画布创建人物节点。
- 删除人物资源。

## 素材库

用于管理普通图片、视频和音频。

素材库跟随项目隔离。上传普通素材时会绑定当前项目，切换项目后只显示当前项目的普通素材；人物角色仍然共享显示。

能力：

- 上传图片/视频/音频。
- 搜索素材。
- 拖拽到画布创建素材节点。
- 放大查看素材。
- 删除素材资源。

## 历史视频

历史视频展示当前项目内的 Seedance 视频生成记录。

能力：

- 查看最近生成的视频任务。
- 放大预览已完成视频。
- 将历史视频重新放入当前画布，作为结果节点查看和整理。

## 删除行为

删除资源库素材会删除本地数据库记录，并清理当前画布引用。

当前版本不主动删除 TOS 中的对象文件，避免误删已被其他项目或历史任务引用的远程资源。
