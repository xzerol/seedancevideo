# 画布工作流

画布基于 React Flow 实现，支持拖拽、缩放、平移、连线和节点选择。

## 节点类型

- `asset`：普通素材节点，可作为下游生成输入。
- `person`：人物角色节点，本质是图片素材，但在 UI 中单独分组。
- `seedream-image`：Seedream 5.0 生图节点。
- `seedance-video`：Seedance 2.0 视频生成节点。
- `bailian-video`：阿里百炼视频生成节点。
- `result-image`：图片生成结果节点。
- `result-video`：视频生成结果节点。

## 预览

- 点击画布中的素材节点可放大查看素材。
- 图片结果可点击预览，视频结果可通过“放大”按钮预览。
- 左侧资源库和历史视频也提供放大入口。

## 连线语义

从素材、人物或结果节点连到生成节点时：

- 目标生成节点会记录输入素材 ID。
- 输入框上方会显示来源卡片。
- Prompt 会维护显式 `@素材名` 引用。

这种方式让输入关系既能被系统读取，也能被用户看见和编辑。

## 新建节点位置

- 如果当前选中了节点，新建生成节点会出现在选中节点右侧附近。
- 如果没有选中节点，新建生成节点会出现在当前视野中心。
- 连续新增节点会轻微错位，避免重叠但不飘到远处。

## @ 素材引用

- 在生成节点输入框输入 `@` 会弹出当前可用素材。
- 选择素材后会在光标处插入 `@素材名`，并绑定到当前节点输入源。
- 候选素材来自当前项目素材库、全局人物库和已连接输入源。

## 删除规则

- 删除画布节点：只从当前项目画布移除节点，并删除相关连线。
- 删除资源库素材：从本地资源库移除，同时清理画布中引用该素材的节点、输入源和连线。
- 删除生成结果节点：只移除当前画布里的结果卡片，不影响已收藏到资源库的素材。
