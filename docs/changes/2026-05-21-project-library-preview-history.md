# 项目素材库、预览与历史视频

## 修改内容

- 画布素材节点、结果图片、结果视频、资源库素材支持放大预览。
- Seedance 视频节点补充视频比例选择。
- 左侧栏新增“历史视频”，展示当前项目的 Seedance 视频生成记录。
- 人物角色库、素材库、历史视频均支持折叠；展开后最多显示 5 条高度，超出滚动。
- 人物角色库改为全局共享。
- 普通素材库改为项目隔离：上传普通素材时绑定当前项目，切换项目后显示不同素材。

## 数据规则

- `libraryType="person"` 的素材作为全局人物角色，不绑定项目。
- `libraryType="asset"` 的素材通过 `ProjectAsset` 绑定当前项目。
- 保存画布时不会清空已上传但未拖入画布的项目素材，避免项目素材库丢失。

## 用户可见效果

- 不同项目看到不同的普通素材库。
- 所有项目共用人物角色库。
- 历史视频可在左侧查看、放大预览，也可重新放回画布。
